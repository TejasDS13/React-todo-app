# React Daily Goal Reminder

A simple and interactive Todo application built with React to practice state management, event handling, and React Hooks.

## Features

- ✅ Add new tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 💾 Save tasks using Local Storage
- 🚫 Prevent empty tasks

## Technologies Used

- React
- JavaScript (ES6+)
- HTML
- CSS

## Concepts Learned

- React Hooks (`useState`, `useEffect`)
- Controlled Components
- State Management
- Rendering Lists with `map()`
- Updating State with `filter()` and `map()`
- Local Storage

## Future Improvements

- Edit existing tasks
- Task categories
- Due dates
- Search functionality
